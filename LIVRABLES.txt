🎨 CARACTÉRISTIQUES DU DESIGN

Palette de Couleurs :
  • Fond : Dégradé noir → bleu foncé (#0a0a0a → #1a1a2e)
  • Or : #d4af37 (titres, accents, liens)
  • Rouge : #c41e3a (sous-titres, bordures)
  • Texte : #e0e0e0 (gris clair)

Typographie :
  • Police : Georgia (serif élégant)
  • Titres : 3em avec effet glow doré
  • Corps : 1.1em, justifié

Effets Visuels :
  • Dégradés or-rouge pour boutons
  • Ombres lumineuses (text-shadow, box-shadow)
  • Transitions smooth (0.3s)
  • Hover effects (translateY, scale)

🚀 DÉPLOIEMENT GITHUB (3 ÉTAPES)

ÉTAPE 1 : Créer le dépôt
  cd /home/ubuntu/sawas-ai-art
  git init
  git add .
  git commit -m "Initial commit - Site SAWAS AI ART"
  git branch -M main
  git remote add origin https://github.com/VOTRE-USERNAME/sawas-ai-art.git
  git push -u origin main

ÉTAPE 2 : Activer GitHub Pages
  1. Aller dans Settings du repository
  2. Section Pages
  3. Source : Deploy from a branch
  4. Branch : main / folder : / (root)
  5. Cliquer sur Save
  6. Attendre 2-3 minutes
  7. Site disponible à : https://VOTRE-USERNAME.github.io/sawas-ai-art/

ÉTAPE 3 : Domaine personnalisé (optionnel)
  1. Settings > Pages > Custom domain : sawas-ai.art
  2. Configurer DNS chez votre registrar (voir README.md)
